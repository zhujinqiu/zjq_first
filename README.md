# 编辑基本语法
我们可以使用下面的方式给我们的文本添加强调的效果<br>
*强调*  (示例：斜体)  
 _强调_  (示例：斜体)  
**加重强调**  (示例：粗体)  
 __加重强调__ (示例：粗体)  
***特别强调*** (示例：粗斜体)  
___特别强调___  (示例：粗斜体)

3、代码块高亮插入代码片段<br>

